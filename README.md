# Crypto Tracker

Application that tracks crypto prices, made with React Native.

## Step 1: Start Metro

First, you will need to run **Metro**, the JavaScript build tool for React Native.

```sh
npm start
```

## Step 2: Build and run your app

With Metro running, open a new terminal window/pane from the root of your React Native project, and use one of the following commands to build and run Android or iOS:

### Android

```sh
# Using npm
npm run android
```

### iOS

```sh
# Using npm
npm run ios
```
